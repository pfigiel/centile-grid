## Description

<!-- SECTION:DESCRIPTION:BEGIN -->

Add a chart library to ui-kit-mobile and build a chart component on top of that.

The chart we need right now is a 2D line chart with standard X and Y axis. It needs to be capable of displaying multiple data series. The chart will be used to present centile growth charts, so most often there will be patient age on X axis and some numeric parameter (height, weight etc.) on the Y axis. There will most often be 5 lines presented at once, one for each centile.

Decide on the chart library during planning, favor solutions that can fulfill the requirements and are also popular, free, with strong community support.
