# @centile-grid/ui-kit-mobile

Shared React Native component library used by the centile-grid mobile app.

## Usage

The package is consumed as a workspace dependency — it is not published to npm.

```json
"@centile-grid/ui-kit-mobile": "workspace:*"
```

## Components

| Component | Description                                 |
| --------- | ------------------------------------------- |
| `Button`  | Themed button wrapping `react-native-paper` |

## Peer dependencies

| Package              | Version  |
| -------------------- | -------- |
| `react`              | ≥ 19.0.0 |
| `react-native`       | ≥ 0.81.0 |
| `react-native-paper` | ≥ 5.0.0  |
